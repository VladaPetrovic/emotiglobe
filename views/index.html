Emotiglobe
See how the world is feeling... read more

This code is experimental. It requires a desktop browser with webGL support e.g. Chrome or Firefox.

Emotiglobe is a visual representation of how the world is feeling right now. The lines drawn in specific geographic locations reveal how people are feeling. The bigger the line is, the more happy expressions we captured from people in that area.

Emotiglobe uses The WebGL Globe, part of Google's awesome Chrome Experiments.
